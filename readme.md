# Калькулятор на HTML/CSS/JavaScript

Простой, но функциональный калькулятор с чистым интерфейсом. Реализован на ванильном JavaScript без использования сторонних библиотек.

## Функционал

- Арифметические операции: сложение, вычитание, умножение, деление
- Смена знака числа (±)
- Вычисление процентов
- Работа с десятичными дробями (точка)
- Очистка ввода (кнопка C)
- Обработка ошибок (деление на ноль)
git remote add origin
## Технические детали

- **HTML** — разметка интерфейса
- **CSS** — стилизация (сетка, анимации, адаптивность)
- **JavaScript** — логика вычислений и обработка событий
